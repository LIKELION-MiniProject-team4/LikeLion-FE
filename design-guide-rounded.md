# 전공 교수 태크 — 둥글둥글 스타일 가이드

기존 와이어프레임(01~07, Desktop-3)을 기준으로, 전체 UI를 더 부드럽고 둥근 느낌으로 바꾸기 위한 스타일 명세입니다.

## 1. Border Radius 스케일

| 요소 | 기존(추정) | 변경 |
|---|---|---|
| 버튼 (로그인, 가입하기, 구매하기 등) | 4~6px | **16px** (완전 pill에 가깝게) |
| 태그/칩 (설명 친절, 과제 적당 등) | 4px | **999px (full pill)** |
| 카드 (교수 카드, 리뷰 카드, 통계 박스) | 6~8px | **20px** |
| 인풋 필드 (이메일, 비밀번호 등) | 4px | **14px** |
| 모달/팝업 (족보 구매 확인창) | 8px | **24px** |
| 프로필/썸네일 이미지 | 사각형 | **원형 (50%)** |
| 상단 검색바 | 6px | **999px (full pill)** |
| 드롭다운 (학과 전체, 학년 등) | 4px | **12px** |
| 배너/광고 박스 | 6px | **18px** |

## 2. 색상

- Primary Blue: `#3B82F6` (버튼, 링크, 강조 텍스트)
- Primary Blue Hover: `#2563EB`
- 태그 배경: `#EFF6FF`, 태그 텍스트: `#3B82F6`
- 별점 옐로우: `#FBBF24`
- 배경: `#FFFFFF` / 카드 배경: `#F9FAFB`
- 테두리: `#E5E7EB` (기존보다 얇게, 1px, 그림자로 구분감 대체)
- 본문 텍스트: `#111827`, 보조 텍스트: `#6B7280`

## 3. 그림자 (Elevation)

둥근 모서리와 함께 각진 테두리 대신 은은한 그림자로 구분감을 줍니다.

- 카드: `box-shadow: 0 2px 8px rgba(0,0,0,0.06)`
- 모달: `box-shadow: 0 8px 24px rgba(0,0,0,0.12)`
- 버튼 hover: `box-shadow: 0 4px 10px rgba(59,130,246,0.25)`

## 4. 컴포넌트별 적용 포인트

**태그/칩** (설명 친절, 과제 적당, 시험 난이도 중 등)
전부 완전한 pill 모양(`border-radius: 999px`)으로, 배경은 연한 블루, 텍스트는 진한 블루. 태그 요약 탭의 숫자 뱃지도 동일한 pill 스타일 유지.

**버튼** (로그인, 가입하기, 구매하기, 자세히 보기)
- Primary 버튼: 채운 파란색 + `border-radius: 16px`, 높이는 기존보다 살짝 키워 여유 있는 느낌
- Secondary 버튼(취소, 자세히 보기): 얇은 테두리 + 같은 radius, 배경은 흰색 또는 연한 회색

**카드** (교수 리스트 카드, 리뷰 카드, 통계 카드)
모서리 20px, 그림자로 입체감. 카드 안쪽 여백(padding)도 넉넉하게 24px 내외로 늘려 답답하지 않게.

**프로필 이미지 / 아이콘 배지**
교수 프로필, 유저 아바타는 완전 원형. 알림/포인트 아이콘 배지도 원형 유지.

**인풋 필드 & 검색바**
검색창은 좌우 완전 둥근 pill 형태(`999px`) + 왼쪽 돋보기 아이콘. 회원가입/로그인 폼의 인풋은 14px 라운드로 카드보다는 덜 둥글게, 버튼보다는 비슷하게.

**모달** (족보 구매 확인창)
가장 큰 라운드(24px) 적용, 그림자 강하게 주어 배경과 확실히 분리. 버튼도 pill 스타일로 통일.

**다이어그램/차트 카드**
레이더 차트를 감싸는 카드도 20px 라운드, 안쪽 여백을 넓혀 차트가 더 여유 있게 보이도록.

**사이드바 (Desktop-3)**
좌측 메뉴 항목 하나하나를 pill 버튼처럼 처리(선택된 "강의 후기" 항목은 연한 블루 배경 + full radius), 로고 박스도 12~16px 라운드.

## 5. 타이포그래피 & 간격

- 카드/버튼의 여백을 전반적으로 넓혀 둥근 형태가 답답해 보이지 않도록 함
- 제목은 semi-bold, 본문은 regular 유지, 자간은 넓히지 않음
- 아이콘도 각진 것보다 둥근 stroke의 line icon 사용 권장 (예: Lucide, Feather 계열)

## 6. 요약

핵심은 "각 요소마다 반경을 하나로 통일하지 않고, 크기가 클수록(모달 > 카드 > 버튼 > 태그) 더 부드럽게 보이도록 라운드 값을 계단식으로 키우는 것"입니다. 여기에 얇은 테두리 대신 그림자를 쓰고, 원형 아바타/pill 태그를 적극 활용하면 전체적으로 훨씬 둥글둥글한 인상을 줄 수 있습니다.
